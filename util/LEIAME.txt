1-) Esse exemplo foi criado apenas usando js, mas pode ser usado criando-se com um modelo webpack se precisar de testes de interface web.

2-) É importante registrar um usuário, se o comando :
#npm whoami
resultar num erro, registre o usuário com :
#npm adduser

3-) O código deve estar num arquivo chamado index.js ou, configure o package.json a propriedade main para apontar para o código.

4-) Depois de fazer a biblioteca e testar, use o 
#npm publish 
para publicar.

5-) Para usar, veja ../using-private-node-module
